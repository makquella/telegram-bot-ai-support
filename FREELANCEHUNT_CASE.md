# SmartFlow AI for Freelancehunt

## Короткое описание проекта

Telegram AI-бот для поддержки, FAQ и внутренних знаний компании. Бот умеет:

- отвечать на текстовые сообщения через LLM
- помнить контекст диалога через Redis
- искать ответы по загруженным PDF, DOCX и TXT через RAG + Qdrant
- обрабатывать голосовые сообщения: STT -> LLM -> TTS
- работать в polling и webhook режимах
- запускаться локально или в Docker

## Что именно реализовано

- архитектура на `aiogram 3.x`
- интеграция LLM через `LiteLLM`
- пользовательская изоляция документов в RAG
- health checks для Redis и Qdrant
- structured logging
- Docker-ready окружение
- unit tests + CI на GitHub Actions

## Бизнес-польза

- можно использовать как AI-консультанта по базе знаний
- подходит для customer support, FAQ-ботов, внутренних ассистентов команды
- снижает нагрузку на менеджеров и поддержку
- позволяет быстро превратить документы клиента в чат-интерфейс
